# WebsiteSeeker
The project is developed in Python and can mail the screenshots of the websites that the user wants from the remote computers.
This project uses cryptography to decrypt the passwords
And right now can only be used in Mozilla Firefox or Thunderbird or Safari
This project works if no master password has been set.
Replace the line 622 & 623 with your gmail account's username and password
The email will be sent by and to this gmail account only
